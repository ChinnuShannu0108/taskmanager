# Task Manager API [project]

A Spring Boot based backend web application that enables users to manage their tasks efficiently. It provides REST APIs to create, update, delete, and retrieve tasks, with MySQL database integration.

## used technologies
- Java
- Spring Boot
- MySQL
- REST APIs
- JWT Authentication
- Spring Security

## what is the use
- User Register
- User Login
- JWT Token Authentication
- Create Task
- View All Tasks
- Update Task
- Delete Task

## How to process
1. Clone the repository
2. Configure MySQL database in application.properties
3. Run TaksmanagerApplication.java
4. Test APIs by using Postman on http://localhost:8080
